1. For Client PC

1.1 Install Client Add-in

Double-click "Client\install.bat" to execute it.

1.2. Uninstall Client Add-in

Double-click "Client\uninstall.bat" to execute it.

1.3 Client Operations

Press WIN + R, type "mstsc" and connect to the server.

2. For Host Server

2.1 Deploy Server App

Copy directory "Server" to the host server.

2.2 Server Operations

(1) Pass URL into "RDPVirtualChannel.Server.exe" via command line parameter.

(2) For example: Press WIN + R, type "cmd", drag or type "RDPVirtualChannel.Server.exe" into cmd, append URL and press Enter. Or do this by programming.
